OpenSquad News?!?!

Our very first article!

Hello and welcome to our very first OpenSquad news article! Expect Special Events notifications, Balance Changes, Community Events and more fun stuff we will be doing!

What is happening with OpenSquad?

OpenSquad is not ending development! I am currently working on our next update which will improve the balance of the game! Expect big changes and more news the following days!

What is OpenSquad?

OpenSquad (Officially created by 8-Bit Hacc) is a private server which aims to adding all of the current content from Squad Busters' until its final update to this version keeping the 1.0 spirit alive!

What will this update bring?

This update will mark the start of serious improvements to OpenSquad! Here are some of them:
1. OpenSquad is now running 24/7 meaning that you can now jump in the game whenever you want!
2. 2 Squaddies (Heavy and Bo) are getting their power levels increased to 10!
3. Some small balance changes before the next update:
       - Bo
DPS (220 -> 250)
Attack Speed (250 -> 200)
       - Heavy
DPS (360 -> 375)

What's next?

Since this update is out of the way let's see what's next on our roadmap!
1. Improve the overall game balance
We have some serious problems with the economy and balance of the game overall. Expect big changes coming to characters and resources in the following months!
2. Better visuals
The current UI and textures are pretty outdated compared to the official game. We will do our best to improve the UI and make the characters look even better as updates go on!
3. New Squaddies and Worlds
We know what you are thinking "Where is all of the content promised?", well, we are working really hard to add everything from the current game so we will share more news soon!
4. Permanent improvements
It is cristal clear that OpenSquad misses a lot of features and game logic. We are working really hard on fixing the game's core issues and make it even better! There are tons of features we want to experiment with (alliances, turbo recharge, new abilities, different upgrade system etc.). We will share more info on that soon!

Closure

Now with all things settled we wish you all the best and hope to see you in the next post! Thank you for supporting us in our journey!
